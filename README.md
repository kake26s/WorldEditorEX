# WorldEditorEX
Jupiter/Nukkit用ワールドエディタ  
  
## 主な機能
- ブロック一括設置(/set [id:meta])  
- ブロック一括削除(/cut)  
- ブロック一括変更(/replace from[id:meta] to[id:meta])  
- コピー(/copy)  
- ペースト(/paste)  
- コピーされたオブジェクトの外部エクスポート(/export [Name])  
- エクスポートされたファイルのインポート(/import [Name]) 
- インポート可能ファイルリストの表示(/objects)  
- 手に持っているアイテムのid確認(/bid)  
- 座標確認(/xyz)
